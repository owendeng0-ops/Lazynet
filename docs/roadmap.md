# Roadmap

## v1.0 Platform Baseline

- OpenWrt-centered runtime.
- Mihomo generated config pipeline.
- Modular rules.
- Config version metadata.
- Healthcheck and update scripts.
- Repeatable build and test commands.
- Shadowrocket and Verge generated outputs.
- Static dashboard shell.

## v1.1 Device Access

- Dashboard device access page.
- Manual proxy endpoint display.
- Current transparent proxy device display.
- Copyable OpenWrt command for changing `LAZYNET_CLIENT_IP`.
- Access checks for service, proxy port, current node, and real exit.

## v1.2 Monitoring

- Netflix node health checks.
- GitHub and AI route diagnostics.
- Node failure reason display.
- Notification adapters.

## v1.3 DNS Layer

- AdGuard Home integration.
- DNS responsibility split between Mihomo and AdGuard Home.
- Better DNS policy testing.
- Better rule testing.

## v1.4 Client Outputs

- Dashboard download links for Shadowrocket and Verge profiles.
- Client profile version display.
- Generated client config validation.

## v2.0 Control Plane

- Confirmed one-click device onboarding.
- Runtime status and historical config comparison.
- One-click update workflow.
- Rollback workflow.
- Dashboard action audit log.
